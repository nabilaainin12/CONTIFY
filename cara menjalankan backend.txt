POST  http://127.0.0.1:8000/api/register
pada Body -> JSON isi dengan
{
    "name": "name",
    "email": "user/admin@gmail.com",(yang ada didataabse)
    "phone": "08123456789",
    "password": "password123"
}

POST  http://127.0.0.1:8000/api/login
pada Body -> JSON isi dengan

{
  "email": "user/admin@gmail.com", (yang ada didatabase)
  "password": "password123"
}

POST
POST  http://127.0.0.1:8000/api/orders
pada Body -> JSON isi dengan (misalnya)

{
  "package_id": 1,
  "title": "Konten Instagram",
  "notes": "Membutuhkan desain feed",
  "booking_date": "2026-07-01"
}

GET   http://127.0.0.1:8000/api/orders
GET   http://127.0.0.1:8000/api/orders/{order}
pada Body -> JSON isi dengan (misalnya)

{
  "title": "Konten TikTok",
  "notes": "Konten promosi terbaru",
  "booking_date": "2026-07-02"
}

PUT   http://127.0.0.1:8000/api/orders/{order}
PUT   http://127.0.0.1:8000/api/orders/{order}/cancel

Catatan: Ganti {order} dengan ID order, misalnya 1, 5, atau ID lain yang ada di database.
